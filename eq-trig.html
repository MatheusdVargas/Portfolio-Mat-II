Equações e Funções Trigonométricas

O que são equações trigonométricas?

Equações trigonométricas são igualdades que envolvem funções como seno, cosseno, tangente, entre outras. Nosso objetivo é encontrar todos os valores de α que tornam a igualdade verdadeira.


Para resolvermos equações trigonométricas devemos isolar a incógnita que está dentro do seno, cosseno ou tangente assim como em equações comuns, se precisarmos passar o sen, cos ou tg para o outro lado da equação utilizamos o arcoseno, arcocosseno ou arcotangente. Durante as aulas só utilizamos seno e cosseno.


Exemplo de uma equação resolvida:

sen ( x ) = 1 2 ⇒ x = arcsen ( 1 2 ) ⇒ x = 30 °


O que são funções trigonométricas?

As funções trigonométricas descrevem fenômenos periódicos como ondas, vibrações, movimentos circulares e comportamentos repetitivos.

Função Seno

A função do seno é caracterizada por uma curva crescente e decrescente sem pontas, como uma onda. Ela começa no 0 e cresce até o 1, volta para o 0 e vai até o -1, retornando ao zero e repetindo o ciclo.


Forma Básica: f(x) = sen(x)
Domínio: todos os reais
Imagem: –1 a 1
Período: 360°

Função Cosseno

A função do cosseno, assim como a do seno é uma onda, mas o seno começa em 1, desce até 0, depois até -1, até 0 novamente e volta ao 1, repetindo o ciclo


Forma Básica: f(x) = cos(x)
Domínio: todos os reais
Imagem: –1 a 1
Período: 360°

Função Tangente

A função tangente é bem diferente das outras, ela é caracterizada como uma linha que possui uma curvatura perto do eixo x mas sobre quase reto para cima conforme de afasta do eixo x. Essa função possui assíntotas verticais que são pontos onde a função não possui valor porque o ângulo cria uma linha paralela a linha da tangente no círculo. Esses pontos podem ser calculados no domínio da função.

f(x) = tg(x) = sen(x)/cos(x)
Domínio: ℝ – {90° + 180°·k}
Imagem: todos os reais
Período: 180°

As funções podem assumir a forma:
f(x) = A + B·sen(Cx + D)

Onde:

  • A → deslocamento vertical
  • B → amplitude
  • C → período
  • D → deslocamento horizontal

O mesmo vale para cos(x).

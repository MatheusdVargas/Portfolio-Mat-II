Equações e Funções Trigonométricas

O que são?

Equações trigonométricas são igualdades que envolvem funções como seno, cosseno, tangente, entre outras. Nosso objetivo é encontrar todos os valores de α que tornam a igualdade verdadeira.


Para resolvermos equações trigonométricas devemos isolar a incógnita que está dentro do seno, cosseno ou tangente. Durante as aulas só utilizamos seno e cosseno.


Exemplo de uma equação resolvida:

sen ( x ) = 1 2 ⇒ x = arcsen ( 1 2 ) ⇒ x = 30 °


O que são?

As funções trigonométricas descrevem fenômenos periódicos como ondas, vibrações, movimentos circulares e comportamentos repetitivos.

Função Seno

f(x) = sen(x)
Domínio: todos os reais
Imagem: –1 a 1
Período: 360°

Função Cosseno

f(x) = cos(x)
Domínio: todos os reais
Imagem: –1 a 1
Período: 360°

Função Tangente

f(x) = tg(x) = sen(x)/cos(x)
Domínio: ℝ – {90° + 180°·k}
Imagem: todos os reais
Período: 180°

Transformações Senoidais

As funções podem assumir a forma:
f(x) = A + B·sen(Cx + D)

Onde:

  • A → deslocamento vertical
  • B → amplitude
  • C → período
  • D → deslocamento horizontal

O mesmo vale para cos(x).
